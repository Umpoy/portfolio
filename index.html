About Me

My passion for coding comes from the joy and frustration I had after years of playing competitive card games. After taking basic online courses of HTML, CSS and JavaScript, I wanted to learn more. With this new obsession of absorbing information, it led me to attend Learning Fuze, an accelerated web development program, where I spent over 700 hours learning and getting hands on experience in a web development environment along with developing several applications.

I have a huge passion for front end web development; specializing in JavaScript and React.js. I always find myself searching for web projects to challenge and improve my skill set. Even as a Front End Developer, I enjoy writing Back End code from time to time using Node.js especially when I’m feeling spicy.
